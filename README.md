# 名言產生器  

這是一個基於Python的名言產生器（Quote Generator），上傳圖片跟輸入名言即可產生名言圖片。

## 功能描述

- 透過選擇圖片並填寫名言及署名，生成帶有名言的圖片。
- 將生成的圖片保存到本地。

## 安裝

使用pip安裝套件：

```bash
pip install -r requirements.txt
```

## 使用
```bash
python quote-generator.py
```
